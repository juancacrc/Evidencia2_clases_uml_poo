# Diagrama Jerarquía Clases

Este documento proporciona un resumen de la jerarquía de clases para el proyecto y muestra las relaciones de herencia entre las diferentes clases utilizadas. También incluye un diagrama UML que ilustra estas relaciones.

## Jerarquía de Clases

- ClaseBase
  - SubClase1
    - SubClase1_1
    - SubClase1_2
  - SubClase2
    - SubClase2_1

## Relaciones de Herencia

- `ClaseBase` es la clase padre de `SubClase1` y `SubClase2`.
- `SubClase1` es la clase padre de `SubClase1_1` y `SubClase1_2`.
- `SubClase2` es la clase padre de `SubClase2_1`.
